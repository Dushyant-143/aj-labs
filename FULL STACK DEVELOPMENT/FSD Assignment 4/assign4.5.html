Key Features of HTML5

1. Semantic Elements

HTML5 introduces new semantic elements like <header>, <article>, <section>, and <footer>, which improve the structure and readability of web pages.

2. Multimedia Support

HTML5 provides built-in support for audio and video using the <audio> and <video> elements, making it easier to embed media without requiring third-party plugins.

3. Canvas API

The <canvas> element allows for dynamic, scriptable rendering of graphics, animations, and games directly in the browser.

4. Local Storage

HTML5 introduces the Web Storage API, including localStorage and sessionStorage, allowing web applications to store data locally within the user's browser without needing cookies.
